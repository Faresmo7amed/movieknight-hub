# MovieKNight 🎬

A movie night planning and review hub built using **pure HTML5**.

The project presents a fictional movie platform where users can explore movie trailers, browse posters, find cinemas, view movie genres, check top picks, learn film terminology, and submit a movie night RSVP form.

## Overview

MovieKNight is a semantic HTML project created to practice building a complete webpage without using CSS or JavaScript.

The goal of this project was to focus on:

- HTML document structure
- Semantic HTML5 elements
- Media elements
- Forms and input types
- Accessibility-friendly markup
- Proper use of HTML attributes

## Features

### 🎥 Featured Trailer

- Embedded HTML5 video player
- Poster image fallback
- Figure and figcaption elements for media descriptions

### 🖼️ Poster Gallery

- Movie poster collection
- Alternative text (`alt`) for images
- Semantic image grouping using `<figure>` and `<figcaption>`

### 📍 Cinema Finder

- Embedded Google Maps location
- Cinema contact information using `<address>`

### 🎭 Movie Categories

- Genre browsing section
- Nested unordered lists for subgenres

### ⭐ Movie Rankings

Different ranking formats using ordered lists:

- Editor ranking
- Critics ranking
- Box office ranking
- Starting number lists
- Reversed countdown lists

### 📖 Film Glossary

A definition list (`<dl>`) explaining movie-related terminology:

- Director's Cut
- Box Office
- Cameo
- Trope
- Reboot
- Franchise
- Post-Credits Scene
- Practical Effects

### 📝 Movie Night RSVP Form

A complete HTML form demonstrating many input types:

- Text input
- Email input
- Telephone input
- URL input
- Password input
- Search input
- Date and time inputs
- Number input
- Range input
- Color picker
- Radio buttons
- Checkboxes
- Select menus
- Multiple select
- Textarea
- File upload
- Meter
- Progress element

## Notes

Some deprecated HTML attributes such as:

- `<center>`
- `leftmargin`

were intentionally used because this project was created without CSS. They were included only to practice HTML structure and basic document presentation without external styling.

In a production project, these should be replaced with modern CSS solutions.

This project is part of my HTML learning journey and focuses on understanding how websites can be structured using semantic HTML before adding styling and interactivity.
